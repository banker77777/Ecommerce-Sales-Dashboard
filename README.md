This project is a full end-to-end Power BI dashboard analyzing ecommerce performance across sales, revenue, orders, customers, categories, payment methods, and state-level insights.
**Key Insights

Monthly revenue and order trends

Top categories by revenue

State-wise contribution using treemap

Payment method distribution

KPI cards showing overall totals

Clean and modern dark-themed design

**Tools & Techniques

Power BI Desktop

DAX Measures

Data modeling

Data cleaning & transformation

Dashboard design

Treemap, Donut Chart, Bar Chart, Combo Chart
